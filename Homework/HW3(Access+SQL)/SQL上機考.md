# 50題抽選5題考試
- 使用sql語法作答
- 可以用access做驗證
- 15分鐘
- 分為5題(查詢、更新、新增建立、刪除、困難題)

## 新增/刪除索引與變更資料表
- 新增索引
``` sql
create index name on table (oID)
```
- 新增欄位
```SQL
alter table add name 資料型別
```
- 刪除欄位
```SQL
alter table drop column name
```
- 刪除索引
``` sql
drop index name on table
```
- 刪除資料表
```sql
drop table name
```
